# Dom Falcone Website

Официальный сайт Дома Фальконе.

## О проекте

Этот сайт посвящен истории, новостям и деятельности Дома Фальконе.

## Запуск локально

Для запуска сайта локально, используйте Python HTTP сервер:

```bash
python3 -m http.server 8080
```

Затем откройте [http://localhost:8080](http://localhost:8080) в браузере.

## Структура

- `homepage/`: Главная страница
- `about/`: О Доме
- `history/`: История
- `enemies/`: Противники
- `news/`: Новости
- `contact/`: Контакты
- `login/`: Вход
- `hymn/`: Гимн
- `photos/`: Фотоотчет
- `assets/`: Ресурсы (CSS, JS, изображения, аудио)
